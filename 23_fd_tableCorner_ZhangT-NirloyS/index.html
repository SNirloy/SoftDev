Table Corner (Sadi and Thomas) here to teach you the foundation of Foundation

Foundation was created to be optimal for websites for mobile browsers. Hence it has specialties in flexboxes and resizing/rescaling to window size.

The Menus of Foundation

  • The normal menu
    a ul tag with the "menu" class
    • The accordions
      • Basic Accordions
          • The requirements of an accordion table
            Needs Foundation Javascript
          • The structure of an accordion table
            • An ul with the "accordion" class
            • data-accordion role="tablist" in opening tag
            • The list element has class="accordion-item is-active" data-accordion-item in the opening tag.
            • Link tag has a bunch of stuff: href="#collapse0" role="tab" class="accordion-title" id="collapse0-heading".
            • The link is followed by a div with id="collapse0" class="accordion-content" role="tabpanel" data-tab-content aria-labelledby="panel1d-heading" data-tab-content.
            • That number after collapse can be changed, but I do not know about the rest.
            • Within that div is the stuff in the box, like this text.
      • Accordion Menus
          • Ul with "vertical menu" class and data-accordion-menu in opening tag.
          • List elements is a link to "#".
          • Elements have ul with class="menu vertical nested is-active".
          • The elements have two parts within
          • First, a link to "#" to nest accordions.
          • Second, a ul with class "menu vertical nested".
          • Inside these uls can be anything, from text to more accordions.
    • Drop Down Menus
      • Single Layer
        • Ul with class="dropdown menu" data-dropdown-menu in the opening tag
        • An empty link to the name of the Menu
        • A ul with the menu class holds all of the elements of the drop down. Using a href="#" will give the text this styling.
      • Inception Any ul with the menu class inside of the dropdown menu class will have the potential to be a dropdown menu by putting another menu ul inside of it. Like above.
